# 🔢 Calculadora de Fatorial em Java
Projeto simples em Java desenvolvido para praticar lógica de programação, utilizando estruturas de repetição, validação de dados e interação com o usuário para calcular fatoriais.

O fatorial de um número é o resultado da multiplicação desse número por todos os seus antecessores até 1.

Por exemplo:

```text
6! = 6 x 5 x 4 x 3 x 2 x 1 = 720
3! = 3 x 2 x 1 = 6
```

## 🎯 Objetivo do Projeto

Este projeto foi criado com o objetivo de praticar conceitos fundamentais da programação em Java, incluindo:
- Validação de entrada de dados
- Estruturas condicionais (`if`)
- Estruturas de repetição (`for`e `while`)
- Interação com o usuário utilizando `Scanner`

## ⚙️ Como executar o projeto
1.  Clonar o repositório:
```bash
git clone https://github.com/Stefanycristina7/fatorial-java.git
```
2.  Entrar na pasta do projeto
```bash
cd fatorial-java
```
3.  Compilar o arquivo Java
```bash
javac Main.java
```
4.  Executar o programa
```bash
java Main
```
